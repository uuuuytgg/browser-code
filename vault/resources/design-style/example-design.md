# Example.com 设计风格分析

## 配色 (Color Palette)

- **背景色**: `#eee` — 浅灰色背景
- **链接色**: `#348` — 深蓝色 (默认色); visited 状态保持一致
- **文字色**: 继承系统默认黑色

整体配色极简，只有背景灰 + 链接蓝两种主色。

## 字体 (Typography)

- `system-ui, sans-serif` — 使用系统默认无衬线字体族
- `h1` 字号 `1.5em`，无额外修饰
- 段落文字无显式字号定义，继承浏览器默认值

## 布局 (Layout)

- **宽度**: `60vw` (视口宽度的 60%)
- **水平居中**: `margin: 0 auto`
- **垂直位置**: `margin-top: 15vh`，让内容块位于视口上部偏中
- 内容区透明度 `opacity: 0.8`，略透明增加柔和感
- 无 padding/border，极简盒模型

## 总体印象

该页面为 IANA 保留的示例域名页面，设计风格**极简 (Minimalist)**，几乎无任何装饰元素。核心设计目标是"清晰展示"而非"美观"，适合作为文档中的中性示例页面。
