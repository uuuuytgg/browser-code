# 02 仓库初始化与目录结构

## 1. 技术选型

```text
语言：TypeScript
包管理：pnpm
运行时：Node.js LTS
构建：tsup / vite
测试：vitest
schema：zod
```

第一版不要上太多复杂工具。

## 2. 初始化命令

```bash
mkdir sidebar-knowledge-agent
cd sidebar-knowledge-agent
git init
pnpm init
```

创建 `pnpm-workspace.yaml`：

```yaml
packages:
  - "apps/*"
  - "packages/*"
```

## 4. 根 package.json

```json
{
  "name": "sidebar-knowledge-agent",
  "version": "0.1.0",
  "private": true,
  "type": "module",
  "scripts": {
    "build": "pnpm -r build",
    "dev": "pnpm --filter @ska/runtime dev",
    "lint": "pnpm -r lint",
    "test": "pnpm -r test",
    "typecheck": "pnpm -r typecheck",
    "rebuild-index": "tsx scripts/rebuild-index.ts"
  },
  "devDependencies": {
    "typescript": "^5.0.0",
    "tsx": "^4.0.0",
    "vitest": "^2.0.0"
  }
}
```
