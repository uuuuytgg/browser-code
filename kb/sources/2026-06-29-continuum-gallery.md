## Summary

Continuum Gallery 是一个基于 Google Photos Picker API 的单页应用，无需后端服务器，提供三种沉浸式浏览模式：瀑布流、轨道滑动和粒子云。纯 Vanilla JS 实现，拥有精致的设计系统（日间/夜间双主题）。

## Key Points

- 三种展示模式：瀑布流（Waterfall）、轨道（Orbit）、粒子云（Sphere）
- 纯 Vanilla JS，无框架，部署在 GitHub Pages
- 完整 CSS 设计系统，含日间/夜间双主题
- Canvas 粒子系统 + 胶片颗粒纹理增强沉浸感
- 支持 Google OAuth 登录和 Google Photos Picker API

## Details

设计系统特色：多层 CSS 背景叠加（网格点阵、渐变光照、径向过渡、纸纹纹理），毛玻璃效果（backdrop-filter: blur），锥形渐变品牌标识。夜间模式重新设计为深空主题而非简单反色。
