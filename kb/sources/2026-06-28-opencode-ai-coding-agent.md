## Summary

OpenCode 是一个开源 AI 编程代理，帮助开发者在终端、IDE 或桌面端编写代码。支持 LSP 自动加载、多会话并行、多种模型提供商（Claude, GPT, Gemini 等），拥有 160K+ GitHub Stars。

## Key Points

- 支持 LSP 为 LLM 自动加载合适的语言服务
- 多会话：同项目并行启动多个代理
- 支持 75+ LLM 提供商，含本地模型
- 隐私优先：不存储用户代码或上下文数据
- 160K GitHub Stars，900 贡献者，7.5M 月活开发者

## Details

OpenCode 提供终端、桌面应用和 IDE 扩展三种使用方式。内置 Zen 精选模型服务，提供经过编程代理基准测试的可靠模型选择。支持 GitHub Copilot 和 ChatGPT Plus/Pro 登录。
