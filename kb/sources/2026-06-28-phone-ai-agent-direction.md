## Summary

UP主"我是HYK"分析手机厂商做 AI Agent 的方向误区：不应追求"一句话搞定一切"的长链路 Agent，而应做短链路、点到为止的 AI 副手。手机的产品形态和用户习惯决定了长链路 Agent 行不通。

## Key Points

- 长链路 Agent 不适合手机：用户习惯、产品形态限制、技术实现困难
- 短链路 Agent 才是正确方向：语音副手做好一两件小事
- 从通知出发而非从屏幕出发，利用已有通知栏场景
- 信任靠一次次"没出错"积累
