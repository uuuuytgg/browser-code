# Model Context Protocol (MCP)

## Metadata

status: active
updated_at: 2026-06-30

## Description

MCP 是 Anthropic 提出的开放标准，用于统一 AI 模型与外部工具/数据的通信方式。核心架构：客户端（LLM/Agent）↔ 服务器（数据库、文件系统、工具）。四大组件：Tools、Resources、Prompts、Context。
