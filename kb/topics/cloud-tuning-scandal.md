# 云控特调与手机测评乱象

## Metadata

status: active
updated_at: 2026-06-30

## Description

云控特调指手机厂商通过云端远程识别设备 IMEI 和应用包名，检测到评测软件时临时拉高频率的一种作弊行为。国家发布《网络测评活动规范》后仍有厂商使用此手段。
