# 投机解码（Speculative Decoding）

## Metadata

status: active
updated_at: 2026-06-30

## Description

投机解码是一种加速 LLM 推理的技术：使用轻量级草稿模型快速猜测多个 token，再由大模型一次性验证。DSpark 是 DeepSeek 在此方向的最新实现，使推理速度提升 60-85%。
