# 世界模型（World Model）

## Metadata

status: active
updated_at: 2026-07-03

## Description

世界模型是一种能够模拟环境反馈的 AI 系统。它通过学习环境状态与观测的演化规律，构建可进行未来预测、内部仿真、规划搜索和行动决策的模型框架。2026 年定义为世界模型爆发元年，AI 从"预测下一个词"跨越至"预测世界下一状态"的认知范式革命。

### 核心阵营

- **Genie 3**（Google DeepMind）— 一张静态图生成可交互 2D 游戏世界
- **Cosmos 2**（NVIDIA）— 物理 AI 世界模型，引入物理对齐 token
- **V-JEPA 2**（Meta/LeCun）— 预测表征而非像素路线，1/50 算力
- **Marble**（World Labs/李飞飞）— 3D 空间智能
- **Qwen-AgentWorld**（阿里）— 35B 开放权重语言世界模型
