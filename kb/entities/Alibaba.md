## 简介

阿里巴巴集团是中国最大的互联网和科技公司之一，业务涵盖电商、云计算（阿里云）、AI、金融科技等领域。2026年7月，因 Claude Code 被曝植入后门，阿里巴巴成为首个公开宣布全面禁用 Claude 全系产品的大型中国企业。
