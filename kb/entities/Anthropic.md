## Description

Anthropic 是一家 AI 公司，以 Claude 系列大语言模型和 MCP（Model Context Protocol）开放标准闻名。MCP 是 Anthropic 提出的用于统一 AI 模型与外部工具/数据通信的开放标准。

2026年6-7月，Anthropic 因其 AI 编程工具 Claude Code 被曝植入隐蔽的中国用户检测机制而陷入信任危机。该机制通过读取时区和代理环境变量识别中国用户，并利用隐写术（修改系统提示词 Unicode 字符）进行标记回传。Anthropic 承认该机制存在并承诺回滚，但该事件已导致阿里巴巴全面禁用其产品，并在全球开发者社区引发对 AI 工具透明度的广泛质疑。
