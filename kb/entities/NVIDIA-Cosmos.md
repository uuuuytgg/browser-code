# NVIDIA Cosmos

## 类型

project

## 简介

NVIDIA 发布的世界模型平台，专门面向物理 AI（机器人/自动驾驶）。Cosmos 2（2026）引入了"物理对齐 token"，让生成的动作与现实世界的力学一致，显著减少 Sim-to-Real 落差。
