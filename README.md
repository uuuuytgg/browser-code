# Sidebar Knowledge Agent

Stage 0 scaffolds the monorepo and shared contracts for a browser-first local knowledge agent.

Current scope:

- Monorepo layout for apps and packages
- Shared schemas and placeholder exports
- Static prompts and tool manifests
- Stage 0 verification via build, test, and typecheck

Not included in Stage 0:

- Runtime tool loop
- Local bridge server
- Vault writes or indexing behavior
- Real model providers
- Downloads, media processing, or MCP execution logic
